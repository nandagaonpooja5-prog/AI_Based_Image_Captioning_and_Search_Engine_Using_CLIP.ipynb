# AI_Based_Image_Captioning_and_Search_Engine_Using_CLIP.ipynb
## Problem Statement
To develop an AI-based semantic image search and captioning system using CLIP for accurate image-text understanding.
## Overview
This project implements an AI-powered semantic image captioning and retrieval system using the CLIP (Contrastive Language–Image Pretraining) model. It generates high-dimensional image and text embeddings to perform multimodal similarity matching. Deep learning and NLP techniques are used for semantic understanding between visual and textual data. The system enables accurate text-to-image retrieval through cosine similarity–based embedding comparison.

- Image Dataset for Captioning & Search
- Input Images with Text Queries
- CLIP Pretrained Embeddings
